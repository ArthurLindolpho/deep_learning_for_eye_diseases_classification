Olá, esse é nosso trabalho de visão computacional, nele utilizamos alguns modelos de deep learning para classificação de doenças oculares, os modelos utilizados foram: VGG19, uma CNN classica e o EfficientNet, estes modelos nos ajudaram muito a explorar e entender 
um pouco mais sobre visão computacional e aprendizado profundo, ao fazer esse trabalho, também aprendemos sobre a arquitetura de cada um desses modelos e suas vantagens e desvantagens. O primeiro passo desse trabalho foi aplicar o pré processamento dos dados, em que 
fizemos um aumento do número de imagens, fazendo rotações, adicionamos filtros para tentar melhorar a precisão dos modelos aplicados, após isso, aplicamos os modelos citados anteriormente para tentar obter melhores resultados. 

Um aprendizado importante que podemos levar desse trabalho é que no campo da visão computacional existem muitos modelos robustos, que possuem uma arquitetura interna bem estruturada, por isso se torna de grande importância fazer testes com esses modelos para ver quais
conseguem obter os melhores resultados.
